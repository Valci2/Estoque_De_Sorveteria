# Sistem de Estoque para Sorveteria

Projeto desenvolvido em C++ para simular o controle de estoque de uma sorveteria, com funcionalidades voltadas para o gerenciamento de produtos, clientes e administração do sistema.

### 1. Requisitos
Antes de rodar, é preciso ter instalado:
- **CMake** 
- **Compilador C++** MinGW
- **Fazer o download dos arquivos compactados (.zip) e descompactar**
  
### 2. Passo a passo

Abra o terminal dentro da pasta do projeto e siga os passos abaixo:

# Cria a pasta onde o programa será compilado
mkdir build

# Entra nessa pasta
cd build

# Gera os arquivos de compilação
cmake ..

# Compila o programa
cmake --build .

Após a compilação terminar, rode o executável:

**Se for windows** : Estoque.exe
**Se for Linux** : ./Estoque
