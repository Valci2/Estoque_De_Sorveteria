Sistema de Estoque de Sorveteria

Nosso projeto consiste em um sistema CRUD de gerenciamento de estoque de uma sorveteria. Nele o usuário do tipo cliente consegue listar os produtos, e fazer solicitações de reserva e, caso for usuário administrador, listar produtos, adicionar produtos, atualizar produtos e remover produtos do sistema.

O projeto foi desenvolvido em C++ para simular o controle de estoque de uma sorveteria, com foco em orientação a objetos.
