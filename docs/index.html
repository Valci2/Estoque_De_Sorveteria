Sistema de Estoque de Sorveteria

Sistema em C++ desenvolvido para a disciplina de Estrutura de Dados Orientada a Objetos, voltado ao gerenciamento de estoque de uma sorveteria, com funções para adicionar, listar, atualizar, remover e reservar produtos conforme o tipo de usuário logado.

Funcionalidades

  • Cadastro e controle de produtos;
  • Gerenciamento de clientes e pedidos;
  • Interface de administração e controle de estoque.
